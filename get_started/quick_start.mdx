## Create your first read request üîç
Dive into blockchain data with your first read request. Fetch data seamlessly with our API by trying the request below.

```cURL Read API example
curl --request POST \
  --url https://api.adamik.io/api/data/state \
  --header 'Content-Type: application/json' \
  --header 'Authorization: <YOUR-API-KEY>' \
  --data '{
  "chainId": "ethereum",
  "address": "0xfc43f5f9dd45258b3aff31bdbe6561d97e8b71de"
}'
```

## Create your first write request üñäÔ∏è
Make your mark on the blockchain. Learn to create write requests to submit transactions or data. Give the following request a try.
It will help you craft a transaction from a Binance account to the address demo.eth on the Ethereum network.

```cURL Write example
curl --request POST \
  --url https://api.adamik.io/api/transaction/encode \
  --header 'Content-Type: application/json' \
  --header 'Authorization: <YOUR-API-KEY>' \
  --data '{
  "transaction": {
    "chainId": "ethereum",
    "mode": "transfer",
    "senders": [
      "0x3f5CE5FBFe3E9af3971dD833D26bA9b5C936f0bE"
    ],
    "recipients": [
      "0xfc43f5f9dd45258b3aff31bdbe6561d97e8b71de"
    ],
    "amount": "1234560",
    "useMaxAmount": false
  }
}'
```
