## Create your first write request üñäÔ∏è

Make your mark on the blockchain. Learn to submit write requests for seamless transaction handling. Whether transferring on Ethereum or Bitcoin, delegating on Cosmos, or claiming rewards on dYdX, the Adamik API simplifies crafting your transactions.

<CodeGroup>

```cURL Transfer Ethereum
curl --request POST \
  --url https://api.adamik.io/api/ethereum/transaction/encode \
  --header 'Authorization: <YOUR-API-KEY>' \
  --data '{
  "transaction": {
    "data": {
      "mode": "transfer",
      "senderAddress": "0x3f5CE5FBFe3E9af3971dD833D26bA9b5C936f0bE",
      "recipientAddress": "0xd8da6bf26964af9d7eed9e03e53415d37aa96045",
      "amount": "12345600000"
    }
  }
}'
```

```cURL Transfer Bitcoin
curl --request POST \
  --url https://api.adamik.io/api/bitcoin/transaction/encode \
  --header 'Authorization: <YOUR-API-KEY>' \
  --data '{
  "transaction": {
    "data": {
      "mode": "transfer",
      "senderAddress": "bc1pdvagvvtvs3k35zxq7hp86x6cgp5kl6np786y50p7dj2m394rhrwq236ryc",
      "recipientAddress": "bc1qekphvuz20qvdhkzywfe29r9vvtwxrszvaxzmqm",
      "amount": "1"
    }
  }
}'
```

```cURL Delegate Cosmos
curl --request POST \
  --url https://api.adamik.io/api/cosmoshub/transaction/encode \
  --header 'Authorization: <YOUR-API-KEY>' \
  --data '{
  "transaction": {
    "data": {
      "mode": "stake",
      "senderAddress": "cosmos1g84934jpu3v5de5yqukkkhxmcvsw3u2ajxvpdl",
      "recipientAddress": "",
      "targetValidatorAddress": "cosmosvaloper1clpqr4nrk4khgkxj78fcwwh6dl3uw4epsluffn",
      "amount": "123"
    }
  }
}'
```

