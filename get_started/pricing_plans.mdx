## How It Works

1. **Free Plan**: Get started without providing any payment information. The free plan gives you 10,000 API calls per month and access to all supported blockchains, with rate limits applied for non-production use.

2. **Pay-As-You-Go Plan**: For production environments, this plan requires a $250 monthly flat fee and comes with 100,000 free API calls. Additional API calls beyond 100,000 are billed according to the dynamic pricing tiers below. This plan is designed to scale with your usage, ensuring cost efficiency as your business grows.

## Pricing Tiers for Pay-As-You-Go Plan

Your cost is recalculated immediately after each API call based on a rolling 30-day total, with prices dynamically adjusting to match the appropriate tier.

## Example: Monthly Cost Calculation

**Example Scenario:**

- **Plan**: Pay-As-You-Go
- **Total API Calls**: 1,200,000 (monthly)

### Calculation:

1. **API Call Charges**:

   - First 100,000 calls:  
     Included in the monthly fee of $250
   - Next 400,000 calls (Tier 1):  
     `400,000 × 0.000711 = $284.40`
   - Next 500,000 calls (Tier 2):  
     `500,000 × 0.000640 = $320.00`
   - Remaining 200,000 calls (Tier 3):  
     `200,000 × 0.000569 = $113.80`
   - **Total API Call Cost**:  
     `$284.40 + $320.00 + $113.80 = $718.20`

2. **Total Monthly Cost**:  
   `$718.20 (API Call Cost) + $250 (Flat Fee) = $968.20`
