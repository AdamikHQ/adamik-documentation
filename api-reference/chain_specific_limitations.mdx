In this page, you'll find all the specific behaviours of each chain. Although Adamik strives to unify all cryptos, not all of them are created equals.

## Bitcoin

XPub format supported:

| Wallet Type   | Format | Supported  |
| ------------- | ------ | ---------- |
| Legacy        | XPub   | Yes        |
| Nested Segwit | YPub   | Yes        |
| Native Segwit | ZPub   | No         |
| Taproot       | ZPub   | No         |

Althought, all addresses format are supported.
