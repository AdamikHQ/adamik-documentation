On this page, you'll find the specific behaviors for each supported blockchain. While Adamik aims to unify all crypto interactions, not all chains operate the same way.

## Bitcoin

Supported XPub formats:

| Wallet Type   | Format | Supported |
| ------------- | ------ | --------- |
| Legacy        | XPub   | Yes       |
| Nested SegWit | YPub   | Yes       |
| Native SegWit | ZPub   | No        |
| Taproot       | ZPub   | No        |

However, all address formats are supported.
